## 💬 프로젝트 소개
- _`오늘은 꿀빨강 하나 건지고 싶은데, 어느 지점으로 운동하러 가는 게 좋을까?`_
- 꿀멩이는 클라이머들이 **꿀문제**를 자유롭게 공유하고 탐색할 수 있도록 만든 웹 서비스입니다.

### 💡 주요 기능
| 기능            | 설명                                   |
| ------------- | ------------------------------------ |
| 🏆 금주의 꿀지점 랭킹 | 난이도별로 가장 많은 꿀문제가 공유된 지점을 랭킹으로 확인     |
| 🧭 꿀문제 공유     | 더클라임 지점/난이도/기술 요소와 함께 꿀문제를 자유롭게 업로드 |
| 🔎 꿀문제 검색     | 원하는 난이도/지점/기술 필터를 조합해 꿀문제를 탐색 및 좋아요로 저장   |
| 🤝 팔로우 및 소통   | 나와 비슷한 수준의 클라이머를 팔로우하고 피드를 통해 교류     |

## ⚙️ 서비스 화면
### 회원가입 페이지
<img width="700" src="https://github.com/user-attachments/assets/8034cdd8-3eaf-4ed5-a37f-12f4e473603e" /><br/>
- **닉네임 중복확인**
  - 기존 사용자가 이미 사용 중인 닉네임은 사용할 수 없습니다.
- **비밀번호 확인**
  - 비밀번호는 대소문자, 숫자, 특수문자를 각각 1개 이상 포함한 8글자 이상이어야 합니다.
  - Spring security를 사용해 암호화하여 저장합니다.

### 로그인 페이지
<img width="700" src="https://github.com/user-attachments/assets/a3539f5d-c519-4cf8-9132-4f4295050d47" /><br/>
- **로그인 인증**
  - 존재하는 이메일인지, 이메일과 비밀번호가 일치하는지 확인합니다.
  - 유효한 정보라면 Access 토큰과 Refresh 토큰을 발급하여 저장합니다.


### 홈페이지
<img width="700" alt="스크린샷 2025-06-07 오전 2 27 43" src="https://github.com/user-attachments/assets/649c3d94-7ad3-4626-8690-50ea1c53a7f4" /></br>
- **지점 추천** 
	- 난이도별로 게시물 수가 많은 지점의 순위를 확인할 수 있습니다.
  - 동점을 기록한 경우 랜덤으로 대표 지점이 결정되며 나머지 지점들은 항아리 안에 표시됩니다.
- **게시글 조회**
	- 사용자들이 등록한 전체 게시글을 조회할 수 있습니다.
	- 한 페이지에서 12개의 게시글이 조회됩니다.
	</br>
<img width="700" alt="스크린샷 2025-06-07 오전 2 37 36" src="https://github.com/user-attachments/assets/ee5d2731-a8e5-4f07-9814-9e23bc342309" /></br>
- **필터링**
	- 사용자는 필터링을 통해 원하는 조건에 해당하는 게시글을 찾을 수 있습니다.
- **검색**
	- 검색창을 통해 유저 혹은 게시글 중 키워드로 검색을 할 수 있습니다. 유저는 닉네임으로,
		게시글은 제목 혹은 내용의 단어로 검색합니다.
	</br>
<img width="700" alt="스크린샷 2025-06-07 오전 2 33 32" src="https://github.com/user-attachments/assets/233037e8-354b-489c-8d7e-26466f85fe84" /></br>
- **게시글 등록**
	- 사용자는 + 버튼을 눌러 게시글을 등록할 수 있습니다.
	- 사용자가 아닌 경우 로그인 화면으로 이동합니다.
- **찜 기능**
	- 사용자는 마음에 드는 게시물에 하트를 눌러 찜을 할 수 있습니다.
	- 사용자가 아닌 경우 로그인을 하겠냐는 모달창이 뜹니다.

### 게시글 상세페이지
<img width="700" alt="스크린샷 2025-06-07 오전 2 30 47" src="https://github.com/user-attachments/assets/3733ee6c-19b4-4ffc-aa39-1b541f2f1b43" /></br>
- **상세 게시글 조회**
	- 사용자는 선택한 게시글의 디테일한 정보를 확인할 수 있습니다.
- **댓글**
	- 사용자들은 게시글에 등록된 댓글들을 조회할 수 있습니다. 한 페이지에 5개의 댓글이 조회됩니다.
	- 사용자는 댓글 작성 혹은 삭제를 할 수 있습니다. 이 때, 본인이 작성한 댓글일 경우에만 삭제 버튼이 보입니다.
- **게시글 수정 및 삭제**
	- 본인이 작성한 게시글일 경우 수정 혹은 삭제할 수 있습니다.

### 마이 페이지
<img width="700" src="https://github.com/user-attachments/assets/107400d7-2336-49bb-ad3e-4da349bfaf51" /><br/>
- **작성한 게시글 조회**
	- 사용자는 본인이 작성한 게시글 전체를 확인할 수 있습니다. 한 페이지에 12개의 게시글이 조회됩니다.
- **찜한 게시글 조회**
	- 사용자는 본인이 좋아요한 게시글 전체를 확인할 수 있습니다. 한 페이지에 12개의 게시글이 조회됩니다.

<img width="700" src="https://github.com/user-attachments/assets/9e8fca3a-1003-4828-95d8-3a9d63b9b1b7" /><br/>
- **회원정보 수정**
	- 사용자는 본인의 프로필 이미지를 원하는 이미지로 설정할 수 있습니다.
  - 사용자는 닉네임과 프로필 소개란을 수정할 수 있습니다.
- **비밀번호 변경**
	- 사용자는 현재 사용 중인 비밀번호를 확인 후, 새로운 비밀번호로 변경할 수 있습니다.

<img width="700" src="https://github.com/user-attachments/assets/d3875506-6808-452c-ab4f-c0f6bee2cc43" /><br/>
- **팔로워/팔로잉 목록 조회**
	- 사용자는 본인을 팔로잉하고 있는 사용자와 본인이 팔로잉하고 있는 사용자의 목록을 조회할 수 있습니다.
  - 팔로우 버튼을 통해 팔로잉 등록 및 취소가 가능합니다.                 |
